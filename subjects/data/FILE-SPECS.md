# Specifications of data files

## [`generated/subjects.csv`](generated/subjects.csv)

File automatically generated by the [`../../.third-parties/get-third-parties.sh`](../../.third-parties/get-third-parties.sh) script.

- `benchmark_name`: Benchmark name
  * factor: DafnyBench

- `program_name`: Program name
  * factor, e.g., 630-dafny_tmp_tmpz2kokaiq_Solution

## `generated/is-verifiable/<Dafny program name>/data.csv`

- `benchmark_name`: Benchmark name
  * factor: DafnyBench

- `program_name`: Program name
  * factor, e.g., 630-dafny_tmp_tmpz2kokaiq_Solution

- `is_verifiable`: Whether the verification of the program runs successfully
  * factor: 0 if it is not verifiable, 1 if it is verifiable

- `runtime`: Time in milliseconds to verify the program
  * numerical, e.g., 345

## [`generated/subjects-whitelist.csv`](generated/subjects-whitelist.csv)

File automatically generated by the following script:
```bash
echo "benchmark_name,program_name" > "generated/subjects-whitelist.csv"
find "generated/is-verifiable/" -type f -name "data.csv" -exec grep "^.*,.*,1,.*$" {} \; | cut -f1,2 -d',' >> "generated/subjects-whitelist.csv"
```

- `benchmark_name`: Benchmark name
  * factor: DafnyBench

- `program_name`: Program name
  * factor, e.g., 630-dafny_tmp_tmpz2kokaiq_Solution

## [`generated/repositories.csv`](generated/repositories.csv)

File manually created from Table 4 in the paper [DafnyBench: A Benchmark for Formal Software Verification](https://arxiv.org/abs/2406.08467), which lists the repository URL of each program in the DafnyBench benchmark.

## [`generated/commits.csv`](generated/commits.csv)

File automatically generated by the [`../scripts/collect-commits.sh`](../scripts/collect-commits.sh) script.

- `commit_url`: the GitHub URL of the commit
  * factor, e.g., https://github.com/aws/aws-encryption-sdk/commit/1d09d63b02310865826db571d4e5607883908efa

- `num_dfy_files`: ???
  * numerical, e.g., 1

- `num_insertions`: ???
  * numerical, e.g., 17

- `num_deletions`: ???
  * numerical, e.g., 3

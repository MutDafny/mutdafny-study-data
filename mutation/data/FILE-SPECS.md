# Specifications of data files

## `generated/scan/data/<mutation operator, BOR|BBR|UOI|UOD|LVR|EVR|LSR|LBI|CIR|SDL>/<Benchmark name>/<Dafny program name>/data.csv`

File automatically generated by the [`../scripts/run-scan.sh`](../scripts/run-scan.sh) script.

- `benchmark_name`: Benchmark name
  * factor: DafnyBench

- `program_name`: Program name
  * factor, e.g., 630-dafny_tmp_tmpz2kokaiq_Solution

- `mutation_operator`: Acronym of the mutation operator
  * factor: BOR, BBR, UOI, UOD, LVR, EVR, LSR, LBI, MCR, CIR, or SDL

- `parsing_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 1

- `plugin_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 3

- `resolution_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 7

- `verification_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 9

- `number_of_targets`: number of targets
  * numerical, e.g., 899

- `scan_time`: total time in milliseconds to run MutDafny's `scan` command
  * numerical, e.g., 7182

## `generated/scan/data/<mutation operator, BOR|BBR|UOI|UOD|LVR|EVR|LSR|LBI|CIR|SDL>/<Benchmark name>/<Dafny program name>/targets.csv`

File automatically generated by the [`../scripts/run-scan.sh`](../scripts/run-scan.sh) script.
The CSV file is headless but organized in three comma separated columns.

- First column represents the line number to be mutated.
  * numerical, e.g., 335

- Second column represents the mutation operator.
  * factor: BOR, BBR, UOI, UOD, LVR, EVR, LSR, LBI, MCR, CIR, or SDL

- Second column represents the mutation.
  * factor, e.g., Ge

## `generated/mut/data/<mutation operator, BOR|BBR|UOI|UOD|LVR|EVR|LSR|LBI|CIR|SDL>/<Benchmark name>/<Dafny program name>/data.csv`

File automatically generated by the [`../scripts/run-mut.sh`](../scripts/run-mut.sh) script.

- `benchmark_name`: Benchmark name
  * factor: DafnyBench

- `program_name`: Program name
  * factor, e.g., 630-dafny_tmp_tmpz2kokaiq_Solution

- `mutation_position`: Line number to mutate
  * numerical, e.g., 335

- `mutation_operator`: Acronym of the mutation operator
  * factor: BOR, BBR, UOI, UOD, LVR, EVR, LSR, LBI, MCR, CIR, or SDL

- `mutation`: The change/mutation
  * factor, e.g., Ge

- `status`: Mutant status
  * factor: invalid, survived, timeout, or killed

- `parsing_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 1

- `plugin_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 3

- `resolution_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 7

- `verification_time`: time in milliseconds to <!-- <TBA> -->
  * numerical, e.g., 9

- `mut_time`: total time in milliseconds to run MutDafny's `mut` command
  * numerical, e.g., 7182
